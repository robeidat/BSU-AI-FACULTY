Case Studies

This course uses several case studies to demonstrate data analysis with AI. Each case study includes data, code examples, and documentation.

Main Case Studies

World Values Survey (WVS)

The 7th Wave of the World Values Survey, cleaned and merged with World Bank GDP data. Used for data documentation and report writing exercises.

Used in: Week 2 (Data Documentation), Week 3 (Report Writing)

Austrian Hotels

A realistic simulated dataset of hotels across Austria for practicing data wrangling and table joins. Contains 8 related tables.

Used in: Week 4 (Joining Tables)

Football Manager Interviews

Post-match interview texts from football managers for sentiment analysis and NLP practice. Includes rating systems for comparison and validation.

Used in: Week 7 (Text as Data I), Week 8 (Text as Data II)

Additional Case Studies

US Earnings Data (CPS)

Current Population Survey data for analyzing earnings patterns, gender gaps, and returns to education.

Used in: Week 6 (From Data to Report)

Employee Commits

Simulated dataset of employee commit activity for exploring AI-generated data analysis comparisons.

Common Support Analysis (R)

Iterative development of R code for causal inference analysis using Claude AI, demonstrating prompt-refine workflow.
